# Support

Use the following public channels for AtomicMemory help:

- Package, adapter, plugin, and docs bugs: open a GitHub issue in this
  repository with a minimal reproduction.
- Feature requests: open a GitHub issue and describe the package, adapter, or
  host workflow you want supported.
- Security reports: do not open a public issue. Follow `SECURITY.md`.
- Questions about published package installation: include the package name,
  package version, Node version, package manager version, operating system, and
  the exact command that failed.

Before opening an issue, check `README.md`, `ROADMAP.md`, package-level READMEs,
and the existing issue list. For host plugins, include the host application
version and install method because compatibility depends on each host's
manifest and runtime behavior.
